# class-meeting
Chat para aulas online

Backend: ASP.NET Core 8 "C#"

Frontend: Angular + TypeScript — para interface, captura de câmera/microfone, integração com WebRTC

WebRTC: para capturar vídeo + áudio no browser e estabelecer conexões peer-to-peer ou via servidor de sinalização

SignalR como canal de sinalização entre front e back

Banco de dados gratuito na nuvem: Oracle Cloud Free Tier
